# Installation
## Create Virtual Environment
```bash
python -m venv venv
```
## Install Dependencies
```bash
pip install -r requirements.txt
```
## Setup API Key
Create an .env file and add the key in it as shown
```bash
GEMINI_API_KEY = "AI********"
```
or directly set the "GEMINI_API_KEY" in command line on the go
## Start the Chatbot
```bash
python main.py
```
